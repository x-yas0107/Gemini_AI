# UIAPduino Family Mart Chime (Ver 1.00)

UIAPduino (CH32V003) を使用し、あの有名な「入店音」を極限まで高音質に再現したプロジェクトです。
単なるメロディ再生にとどまらず、物理的なベルの減衰をシミュレートする独自のエンベロープ論理を実装しています。

---

## 🚀 概要

このプロジェクトは、かつて PIC12F675 用にアセンブラで書かれたロジックをベースに、32bit RISC-V マイコンである CH32V003 へ移植・拡張したものです。
デジタル特有の「カクつき」や「継ぎ目のノイズ」を徹底的に排除し、耳に優しい「97点の再現度」を追求しました。

## ✨ 特徴

- **Phase-Coherent Switching (位相維持)**:
  同音が続く場合、タイマーのカウンタをリセットせずに周期だけを更新。波形の連続性を保つことで、音色変化時のクリックノイズをゼロにしました。
- **Global Decay Slope Logic (グローバル減衰スロープ)**:
  音を跨いで減衰の周期を管理。関数の継ぎ目（i=0）で発生しがちな不自然な再アタック（音量のバンプ）を物理的に防止します。
- **3-Note Tie Optimization (3音連続タイ最適化)**:
  「ファ・ファ・ファ」と続くパートにおいて、一つの長いなだらかな減衰坂道を作る専用アルゴリズムを搭載。
- **Soft Attack**:
  発音開始直後に極短時間の低音量域を挟むことで、金属が叩かれる瞬間の柔らかさを演出。

## 🛠 ハードウェア構成

- **MCU**: CH32V003 (UIAPduino 環境)
- **Clock**: 48MHz
- **Input**: PC5 (タクトスイッチ / 内部プルアップ)
- **Output**: PC0 (PWM出力 / TIM2_CH3)
- **Sounder**: 圧電サウンダ または スピーカー（要抵抗）

### ピンアサイン
| 機能 | ピン番号 | 備考 |
| :--- | :--- | :--- |
| PWM Output | **PC0** | TIM2 Channel 3 / AF Push-Pull |
| Switch Input | **PC5** | Active Low / Internal Pull-up |
