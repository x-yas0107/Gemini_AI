【概要】
UIAPduinoの動画再生プロジェクト用に作成された専用バイナリファイル（.bin）を、
PCの標準プレイヤーで再生可能な「MP4動画ファイル（映像＋音声）」に復元する専用デバッグツールです。

マイコン（CH32V003など）やEEPROMにデータを書き込む前に、PC上で
「正しいドット絵になっているか」「音声とズレていないか」をサクッとプレビュー確認できます。

【主な機能】
* 完全なプレビュー再生: マイコンの挙動をシミュレートし、映像と音声を結合したMP4を全自動で生成します。
* 見やすい自動拡大: 128x64の極小OLEDサイズを、PCモニタで確認しやすいように自動で4倍（512x256）に拡大して出力します。
* 高音質化・ノイズ対策: マイコン特化の特殊な音声データ（15.36kHz / 8bit）を、PC規格（44.1kHz / 16bit）へと内部で自動補正。Windows標準のメディアプレイヤーでもノイズレスのクリアな音で再生できます。

【必要な環境 (動作要件)】
1. Python がインストールされていること。
2. FFmpeg がシステムにインストールされ、パス(Path)が通っていること。
3. 以下のPythonライブラリをインストールしてください。
   コマンドプロンプト（またはターミナル）で以下を実行します：
   pip install opencv-python numpy pydub

【使い方】
1. 動画コンバータで作成したバイナリファイル（例：output.bin）を、このスクリプトと同じフォルダに置きます。
2. 本スクリプトを実行します。
3. 処理が完了すると、同じフォルダに「preview.mp4」という動画ファイルが生成されます。
4. ダブルクリックして、Windows標準の動画プレイヤー等で再生結果を確認してください。
