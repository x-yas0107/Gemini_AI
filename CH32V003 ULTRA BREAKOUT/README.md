# CH32V003 ULTRA BREAKOUT - The 8-Pin Legend

## 【プロジェクト概要】
**「10円マイコン、8つのピン。そこに、アーケードの魂を凝縮する。」**
わずか $0.10$ ドルの RISC-V マイコン「CH32V003J4M6」と SSD1306 OLED を組み合わせ、極上の「ヌルヌル感」と「操作性」を追求したブロック崩しゲームです。
「動くだけ」のコードとは一線を画す、レジスタ直接制御による最適化の極致を目指しました。

---

## ⚡ 究極のリソース効率 (Resource Efficiency)
「1バイトの無駄も許さない」というエンジニアの執念が、驚異的なスリム化を実現しました。

| メモリ種別 | 使用量 (V1.14 時点) | 全体容量に対する割合 |
| :--- | :--- | :--- |
| **Flash (保存領域)** | **4,560 Bytes** (text+data) | **27.8%** (16KB 中) |
| **RAM (作業領域)** | **336 Bytes** (data+bss) | **16.4%** (2KB 中) |

* **オンザフライ描画:** 1KB を要する画面バッファを一切持たず、OLED のページごとにリアルタイム計算・転送を行うことで RAM 消費を約 1/3 に抑制しています。
* **余白の美:** Flash の約 72%（11KB 以上）が空き領域であり、更なる機能拡張が可能な「余裕」を維持しています。

---

## 🎮 快適さへのこだわり (Performance & Feel)
ChatGPT などの AI が生成する標準ライブラリ依存のコードに対し、本プロジェクトは「快適さ」で圧倒します。

* **60fps 級の滑らかさ:** レジスタレベルの I2C 制御により、通信オーバーヘッドを極限まで削減。ボールの軌道にカクつきを許しません。
* **リアル・パドル・ローンチ:** ボールはパドルの上に鎮座し、プレイヤーの合図で射出。空中から唐突に始まる「ズル」を廃止しました。
* **クイック・パドル:** ボリュームの中央付近（$\pm 60^\circ$）で全画面をカバーする高感度設定。手首の返しだけで弾き返すアーケード感覚を再現。
* **物理判定の厳密化:** ボールのめり込みによる「突き抜け（トンネル現象）」を、座標強制復帰アルゴリズムにより完全に排除しました。
* **カスタムフォント:** 3x5 ドットの制約下で、左肩のラインが繋がった力強い「S」や、美しい「G」の造形を独自に定義。

## 🛠 I/O マップ (Hardware Configuration)

### 1. 8-Pin Raw Chip 版 (V1.14)
CH32V003J4M6 本来のピン数をフル活用した、最小構成の仕様です。


| Pin Number | Port/Pin | Function | Hardware |
| :--- | :--- | :--- | :--- |
| **Pin 1** | **PA1** | **SOUND** | Buzzer (Square wave) |
| **Pin 8** | **PD1** | **START** | Momentary Switch (Internal Pull-up) |
| **Pin 4** | **PC4** | **PADDLE** | 10k Potentiometer (ADC Channel 2) |
| **Pin 5** | **PC2** | **I2C SCL** | SSD1306 OLED (SCL) |
| **Pin 6** | **PC1** | **I2C SDA** | SSD1306 OLED (SDA) |

### 2. UIAPduino / Arduino 版 (V1.15)
配線の取り回しを考慮し、基板の片側に I/O を集約した「プロ仕様」の配置です。


| Arduino Pin | Port/Pin | Function | Hardware |
| :--- | :--- | :--- | :--- |
| **D1** | **PA1** | **SOUND** | Buzzer |
| **D2** | **PC0** | **START** | Switch (Pull-up) |
| **D3** | **PC1** | **I2C SDA** | OLED (SDA) |
| **D4** | **PC2** | **I2C SCL** | OLED (SCL) |
| **D6 / A2** | **PC4** | **PADDLE** | Potentiometer (ADC) |
