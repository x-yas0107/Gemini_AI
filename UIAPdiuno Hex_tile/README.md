# UIAPduino OLED Hex_tile (うねうねパターン・ジェネレーター)

六角形（ヘックス）グリッドと3次ベジェ曲線（Cubic Bezier）を組み合わせた、有機的で波打つような「うねうね」パターンを描画するジェネレーティブアート・プログラムです。
UIAPduino (CH32V003) の厳しいメモリ制限（RAM 2KB）環境下において、ベクターグラフィックスのリアルタイム演算とオンザフライ描画を実現しています。

- **開発日:** 2026年6月4日
- **開発者:** yas & AI Assistant (Gemini)
- **バージョン:** 0.19

## 特徴
* **3次ベジェ曲線 (Cubic Bezier) による滑らかな描画:** 単純な円弧ではなく、動的な制御点を用いた3次ベジェ曲線演算により、美しく滑らかなリボン状のラインを生成。
* **隣接タイル衝突防止ロジック (Anti-collision):** 同じ向きのタイルが塊になる（clumping）のを防ぐアルゴリズムを搭載。全体がより有機的で変化に富んだ、迷路のような模様に仕上がります。
* **高度な乱数生成 (Advanced RNG Seeding):** ADCの未接続ピンから拾う環境ノイズに加え、人間がスイッチを押している時間（`micros()`）をシード値に合成することで、完全に予測不能なパターンを生み出します。
* **極限のメモリ最適化:** 画面全体を一度に記憶せず、8分割したページごとにベクター演算と描画を行う（128バイトのバッファのみ使用）ことで、RAM不足を完全に解消。

## UIAPduino (CH32V003) I/Oマップ
UIAPduino基板への接続は以下のピンを使用してください。

| 機能 | UIAPduino ピン番号 | マイコン内部ピン | 備考 |
| :--- | :---: | :--- | :--- |
| **SDA** | `3` | PC1 | OLEDの SDA へ接続 |
| **SCL (SCK)** | `4` | PC2 | OLEDの SCL (SCK) へ接続 |
| **SWITCH**| `A2` | PC4 | タクトスイッチを接続 (内部プルアップ) |
| **VDD** | `5V`| - | OLEDの VDD へ接続 |
| **GND** | `GND` | - | OLEDの GND へ接続 |

## 操作方法
1. 電源を入れると、完全ランダムな「うねうね」パターンが生成されOLEDに表示されます。
2. タクトスイッチ（ピン番号: A2 / PC4）を押すことで、押下時間の揺らぎを含んだ新しいシード値が計算され、パターンが瞬時に再生成されます。
