# UIAPduino - I2C Commander (GemOS V1.25)

16KBの極限ロマン。プログラム（C言語）を一切書かずに、I2Cデバイスを直接操作・監視できる究極のハードウェア・ハッキングツールです。

## 概要 (Overview)
UIAPduinoは、フラッシュメモリわずか16KBのマイコン上に構築された独自OS（GemOS）と、I2Cデバイス制御アプリ（I2C_Commander）の統合環境です。
外部EEPROMに「デバイス辞書」と「コマンドシーケンス（SEQ）」を保存することで、**マイコン側のプログラムをコンパイルし直すことなく**、PCからのテキスト送信（コピペ）だけで新しいセンサーやディスプレイを接続して即座に遊ぶことができます。

## 主な機能 (Features)
- **SCN (I2Cスキャナ):** バス上のデバイスを自動検出し、EEPROMの辞書と照合してデバイス名を表示。
- **DMP (リアルタイムダンプ):** 指定したI2Cデバイスのレジスタを500ms間隔で画面にリアルタイム表示。
- **MON (モニターモード):** センサーの生データを指定の計算ルール（SEQ 15）に従い、マイナス表記対応の実数（10進数）としてOLEDにリアルタイム表示。
- **SEQ (シーケンス実行):** 事前に登録した初期化コマンドなどをワンクリックでターゲットに送信。
- **PC LINK MODE:** シリアル通信経由でPCからデバイス辞書やコマンドを直接EEPROMへ流し込める特殊モード。

## ハードウェア構成 (Hardware Requirements)
- **MCU:** CH32V003 などの16KB Flashマイコン
- **Display:** 128x64 OLED (SSD1306 / デフォルトI2Cアドレス: 0x3C または 0x78)
- **Storage:** AT24Cシリーズ EEPROM (デフォルトI2Cアドレス: 0x50)
- **Input:** アナログジョイスティック (ADC 2ch + Push Switch)
- **Switch:** PC LINK MODE切り替え用トグルスイッチ (PC3ピン)

## 使い方 (How to Use)
1. **SCNモード:** 起動後、トップメニューの「SCN」をクリックしてI2Cデバイスを検索。見つかったデバイスを選択（SEL）します。
2. **COMモード:** デバイスを選択後「COM」メニューへ入ります。
    - `RED` / `WRT`: ジョイスティックで指定した単一レジスタの読み出し・書き込み。
    - `DMP`: メモリマップ（レジスタの一覧）をリアルタイムダンプ。
    - `SEQ`: 登録済みコマンド群から選んで送信。
    - `MON`: モニターモード（事前にPC LINK MODEで `SEQ 15` の登録が必要です）。

## PC LINK MODE と シリアルコマンド
PC3ピンをGNDに落とす（スイッチON）と、マイコンが「PC LINK MODE」に移行し、OLED画面の文字が中央に配置されます。
この状態で、PCのシリアルモニタ（ボーレート: 115200bps）から以下の専用コマンドを送信して、ノーコードでシステムを構築します。

### Wコマンド (デバイス辞書登録)
新しいI2Cデバイスをシステムに認識させます。
*書式:* `W,スロット(00-30),I2Cアドレス(Hex),デバイス名(最大14文字)`
*例 (BMP180をスロット03に登録):* `W,03,77,BMP180`

### Sコマンド (シーケンス / MONルール登録)
デバイスに送信するコマンドや、MONモードでの表示ルールを登録します。
*書式:* `S,スロット(00-30),SEQ番号(00-15),表示名(8文字),送信バイト数(Hex),ペイロード(Hex)...`
*例 (スロット03のMONモード表示ルールをSEQ15に登録):* `S,03,15,MON_RULE,04,F6,01,20,50`
> **The Magic of MON Mode:**
> `SEQ 15` はMONモード専用の特別スロットです。ここに「読み取るレジスタ」「データ長」「画面に出すラベル文字」を定義するだけで、プログラムの改変なしにセンサー数値を画面化できます。

### Rコマンド (辞書ダンプ)
現在のEEPROMに登録されているデバイス辞書（スロット一覧）をシリアルモニタに出力します。
*書式:* `R`

### Fコマンド (フォーマット)
スロット03〜30のユーザー領域を安全に初期化します。
*書式:* `F,1234`
