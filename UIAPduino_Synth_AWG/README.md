# UIAPduino_Synth_AWG (Ver 1.00)

## 概要 (Overview)
フリスクサイズに収まる手書きデジタルシンセサイザー 兼 AWG（任意波形発生器）です。
小さなOLED画面上でツマミを回して「点（アンカー）」を打ち、それらを直線や滑らかな曲線で結ぶことで、自分だけのオリジナルの音の波形を自由に直感的に作り出すことができます。

## 開発情報 (Developer Info)
* **開発者:** yas & Gemini
* **リリース日:** 2026年3月29日
* **名前の由来:** ハードウェアプロジェクト「UIAPduino」シリーズを冠し、Synthesizer（シンセサイザー）と AWG（Arbitrary Waveform Generator = 任意波形発生器）の機能を融合させたことから命名されました。

## 特徴 (Features)
* **最大20点のアンカー描画:** 画面上に最大20個の点を自由に配置可能。
* **選べる補間モード:** 打った点同士を「直線（カクカク）」で結ぶか、「スプライン曲線（なめらか）」で結ぶかを選択できます。
* **8つのプリセット内蔵:** サイン波、三角波、矩形波などの基本波形から、減衰波形（DAMP）や重畳波形（DUAL）まですぐに呼び出せます。
* **直感的な操作UI:** アナログのブレ（ノイズ）を完全にカットする不感帯フィルターを搭載。Y軸のツマミは「右に回すと上に移動する」直感的な仕様です。
* **ハードウェア保護:** 描画時の演算エラーやメモリ破壊を極限まで排除した、絶対にフリーズしない強固な専用エンジンで駆動します。

## 操作方法 (How to Use)
### メニュー画面 (MENU Mode)
* **Xツマミ (Board 0):** 音量の調整
* **Yツマミ (Board 1):** 音程（ピッチ）の調整
* **Xボタン / Yボタン:** プリセット波形の切り替え（1〜8）
* **ENTERボタン:** 決定（プレイ画面へ移行）

### プレイ画面 (PLAY Mode)
波形を実際に手書きするモードです。
* **Xツマミ:** カーソルを左右（X軸）に移動
* **Yツマミ:** カーソルを上下（Y軸）に移動（右回しで上昇）
* **ENTERボタンの操作（長さで機能が変わります）**
  * **カチッと短押し (50ms〜):** 現在のカーソル位置に点を打つ
  * **1秒長押し (1000ms〜):** 打った点同士を「直線」で結んで波形を生成・発音
  * **2秒長押し (2000ms〜):** 打った点同士を「なめらかな曲線」で結んで波形を生成・発音
  * **3秒長押し (3000ms〜):** メニュー画面へ戻る（波形リセット）

## ハードウェア構成 / ピンアサイン (Pinout)
UIAPduinoの物理Board番号（0〜10）に完全対応しています。

| Board番号 | マイコンピン | 役割 | 接続機器 |
| :--- | :--- | :--- | :--- |
| **Board 0** | PA1 (ADC) | X軸操作 / 音量 | 可変抵抗 (ボリューム) |
| **Board 1** | PA2 (ADC) | Y軸操作 / 音程 | 可変抵抗 (ボリューム) |
| **Board 3** | PC1 (I2C) | SDA通信 | OLEDディスプレイ |
| **Board 4** | PC2 (I2C) | SCL通信 | OLEDディスプレイ |
| **Board 6** | PC4 (PWM) | オーディオ出力 | スピーカー / アンプ |
| **Board 7** | PC5 (DIG) | ENTERボタン | プッシュスイッチ |
| **Board 8** | PC6 (DIG) | Xボタン (MENU用) | プッシュスイッチ |
| **Board 9** | PC7 (DIG) | Yボタン (MENU用) | プッシュスイッチ |
| **Board 10**| PD0 (DIG) | 状態表示LED | LED |
