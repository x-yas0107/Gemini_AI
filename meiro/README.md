# UIAPduino OLED Maze Generator

Hunt-and-Killアルゴリズムによる、ループのない一筆書きの超複雑迷路ジェネレーターです。
UIAPduino (CH32V003) の極限のハードウェア制限（RAM 2KB / ROM 16KB）を完全に突破するため、論理データの超圧縮と「Enclosed Box（完全密閉枠）」ロジックによるオンザフライ描画を実装しています。

- **開発日:** 2026年6月4日
- **開発者:** yas & AI Assistant (Gemini)

## 特徴
* **限界突破のメモリ最適化:** 1024バイトのVRAM配列を廃止し、わずか512バイトの圧縮論理データとオンザフライのリアルタイム描画エンジンでRAM制限をクリア。
* **Flash容量の極小化:** 重い描画関数を廃止し、IN/OUTラベルを極小のビットマップデータ化。初期化コマンドも配列ループ化することで16KBのROM制限をクリア。
* **Hunt-and-Kill法:** 直線の少ない、非常にクネクネと入り組んだ難易度の高い迷路を生成。
* **Enclosed Boxロジック:** IN/OUTの文字背景処理で壁に穴が開く（ループが発生する）バグを、文字枠の「柱化」と「単一接続穴」により数学的に完全防止。100%一筆書きの迷路を保証。
* **白潰れ防止:** 通路幅3pxに対し、壁幅1pxの描画を行うことでOLED特有の白潰れを解消し、視認性を向上。

## UIAPduino (CH32V003) I/Oマップ
UIAPduino基板への接続は以下のピンを使用してください。

| 機能 | UIAPduino ピン番号 | マイコン内部ピン | 備考 |
| :--- | :---: | :--- | :--- |
| **SDA** | `3` | PC1 | OLEDの SDA へ接続 |
| **SCL (SCK)** | `4` | PC2 | OLEDの SCL (SCK) へ接続 |
| **SWITCH**| `A2` | PC4 | タクトスイッチを接続 (内部プルアップ) |
| **VDD** | `5V`| - | OLEDの VDD へ接続 |
| **GND** | `GND` | - | OLEDの GND へ接続 |

## 操作方法
1. 電源を入れると、完全ランダムな迷路が生成されOLEDに表示されます。
2. タクトスイッチ（ピン番号: A2 / PC4）を押すことで、ランダムシードが再計算され、新しい迷路が瞬時に再生成されます。
