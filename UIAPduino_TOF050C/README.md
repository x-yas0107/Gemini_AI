# UIAPduino_TOF050C (V1.00)

**高精度ToF距離センサー & 空間プロファイル・スキャナー**

`UIAPduino_TOF050C` は、超安価なRISC-Vマイコン **CH32V003** と ToFセンサー **VL6180X (TOF050C)** を組み合わせた、グラフィカルな距離計測デバイスです。単なる距離測定にとどまらず、センサーの放射状視野（FoV）を空間の「プロファイル」として捉え、物体の動きや形状をリアルタイムに可視化します。

---

## 🚀 主な特徴

* **リアルタイム・ヒストリーグラフ**: 過去128サンプルの距離推移をSSD1306 OLED上に鮮明に描画。物体の表面形状や動きの軌跡を視覚化します。
* **方向認識ジェスチャー判定**: 
    * 放射状の検知エリアへの物体進入（Entry）と離脱（Exit）を個別に検知。
    * 接近時には `←`、離脱時には `→` のロング矢印をスマートに表示します。
* **高精度2点キャリブレーション**: 50mmと150mmの基準点による校正機能を搭載。個体差を吸収し、常に正確な距離を算出します。
* **高速描画最適化**: I2C 400kHz通信とレンダリングロジックの最適化により、滑らかなグラフ表示（従来比 約1.5倍の体感速度）を実現。

## 🛠 ハードウェア構成

| コンポーネント | モデル / 仕様 | 役割 |
| :--- | :--- | :--- |
| **MCU** | CH32V003 (RISC-V) | メイン制御・演算 |
| **Sensor** | VL6180X (TOF050C) | ToF方式 距離測定 |
| **Display** | SSD1306 (I2C 128x64) | UI・グラフ描画 |
| **Buttons** | タクトスイッチ x2 | キャリブレーション用 |

### ピンアサイン (CH32V003)

| Pin | 接続先 | 説明 |
| :--- | :--- | :--- |
| **PC1** | SDA (OLED & Sensor) | I2C データ |
| **PC2** | SCL (OLED & Sensor) | I2C クロック |
| **PD5** | Calibration SW (50mm) | プッシュスイッチ（内部プルアップ） |
| **PD6** | Calibration SW (150mm) | プッシュスイッチ（内部プルアップ） |

## ⚙ キャリブレーション手順

精度の高い計測を行うため、初回起動時やセンサー位置変更時には以下の校正を行ってください。

1. **50mm点の設定**: センサーから正確に50mmの位置に平らな板を置き、**PD5** に接続されたボタンを押します。画面に `50ok` と表示されれば成功です。
2. **150mm点の設定**: 同様に150mmの位置に板を置き、**PD6** に接続されたボタンを押します。画面に `150ok` と表示されれば校正完了です。

---

## 📂 インストール

1. WCHの標準ツールチェーン、またはArduino環境（CH32V003対応版）を用意します。
2. `Wire` ライブラリが有効であることを確認してください。
3. `UIAPduino_TOF050C.ino` をマイコンに書き込みます。
