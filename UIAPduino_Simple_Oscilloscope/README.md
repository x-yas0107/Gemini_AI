# UIAPduino Simple Oscilloscope (V1.17)
**CH32V003 (16KB ROM) 限界突破の極小・高機能デジタルオシロスコープ**

UIAPduino（CH32V003F4P6）とOLEDディスプレイ（SSD1306）、そしてわずかな外付け部品だけで構成される、極限まで最適化されたデジタルオシロスコープです。
I2Cのダイレクト制御と24MHzの限界駆動により、16KBという極小のROM容量の中に「シングルショット機能」や「マニュアルホールド機能」を備えた実用的な測定器を実装しました。

## ✨ 主な機能
* **超高速・軽量描画:** OLEDへのダイレクトI2C制御による一筆書き描画。
* **トリガー機能:** トリガーレベル調整（5段階）およびエッジ選択（立ち上がり/立ち下がり）。
* **オートセンタリング (AC/DC対応):** 波形を自動で画面中央に捉える追従機能。
* **シングルショット (単発波形捕獲):** 突発的なパルス波形を待ち伏せして画面に固定表示（※DCモード限定）。
* **マニュアルホールド:** 観察中の波形を任意のタイミングでフリーズさせる一時停止機能。
* **オートゼロ・キャリブレーション:** 基準電圧と入力を直結させたワンタッチ補正。

## 🔌 ピン配置 (Hardware Setup)
* **PC1 (Pin 9):** I2C SDA (OLED)
* **PC2 (Pin 10):** アナログ入力 A2 (TL431 2.49V 基準電圧)
* **PC4 (Pin 12):** アナログ入力 A0 (オシロスコープ入力プローブ / 100kΩ・10kΩ 分圧)
* **PC5 (Pin 13):** SW1 (FUNボタン / 長押しでオートゼロ)
* **PC6 (Pin 14):** SW2 (UP / ＋ボタン)
* **PC7 (Pin 15):** SW3 (DOWN / －ボタン)

## 🎮 基本操作とUIガイド

操作は3つのボタン（FUN, UP, DOWN）で行います。
**FUNボタン (SW1)** を短く押すたびに、画面下部に並んだ設定項目（アンダーラインでハイライト表示）が切り替わります。
設定したい項目にアンダーラインを合わせ、**UP (SW2) / DOWN (SW3)** ボタンで値を変更します。

### 1. 電圧スケール (V/div)
* 画面左下の数値。波形の縦軸（電圧）の拡大・縮小を行います。

### 2. 時間スケール (ms/div)
* 画面下部、左から2番目の数値。波形の横軸（時間）の拡大・縮小を行います。

### 3. トリガーレベル調整
* 画面下部の中央左側。UP/DOWNボタンでトリガーの検知レベルを上下5段階に微調整できます。

### 4. 動作モード切替 (Run / Single / Hold) ★重要★
* 画面下部の中央右側のアルファベット（`R`, `S`, `H`）で、オシロスコープの動作モードを制御します。UP/DOWNボタンでロータリー式に切り替わります。

  * **`R` (Run - 連続描画):** 通常のオシロスコープ動作です。常に波形をサンプリングし続けます。
  * **`S` (Single Shot - 単発待機):** 波形がトリガー条件を満たすまで画面を待機させます。波形を捕獲すると自動的に `H` に移行し、波形をフリーズさせます。
  * **`H` (Hold - マニュアルホールド):** 現在の画面を強制的に一時停止（フリーズ）します。波形をじっくり観察したい時に使用します。

> **⚠️ 【ハードウェア・フェイルセーフ仕様】**
> 本機はACカップリングコンデンサの物理的飽和を防ぐため、**ACモード選択時は `S` (Single Shot) モードが自動的にスキップされます。** ACモードでは `R` (連続) と `H` (手動停止) のみを行き来し、安全かつ正確な測定を保証します。単発パルスの捕獲（Sモード）は、DCモードでご利用ください。

### 5. トリガーエッジ選択 (Rise / Fall)
* 画面右下の矢印アイコン。波形が「立ち上がる時（上向き矢印）」か「立ち下がる時（下向き矢印）」のどちらでトリガーをかけるかを選択します。

### 6. AC/DC カップリング切替
* 画面右下の文字。AC（交流）とDC（直流）モードを切り替えます。

## 🛠 便利な機能
**【オートゼロ・キャリブレーション (基準電圧補正)】**
測定プローブを外部の信号から開放した状態で、**A0ピン（入力）とA2ピン（基準電圧）をショート（短絡）**させてください。
その状態で **FUNボタン (SW1) を1秒以上長押し**すると、現在のシステムの実基準電圧を「ゼロV（センターライン）」としてマイコンが自動で読み込み、正確に補正・記憶します。
