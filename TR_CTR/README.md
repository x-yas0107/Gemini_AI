低価格なRISC-Vマイコン **CH32V006** を使用した、高精度・低ノイズなトランジスタ特性測定器（カーブトレーサー）プロジェクトです。

## 🌟 プロジェクトの概要

このプロジェクトは、最小限の部品構成で最大限の測定精度を引き出すことを目的としています。
当初、出力バッファとしてオペアンプを導入していましたが、デバッグの結果「オペアンプを介さないマイコン直結駆動」が最も0V付近の精度が高く、発振リスクもゼロであることを突き止めました。

### 主な特徴
- **究極のシンプル構成**: オペアンプを排除し、マイコンのPWM + RCフィルタで直接トランジスタを駆動。
- **高精度サンプリング**: 187.5kHzの高速PWMによる低リップル化と、ADCの「空読み」＋「平均化」ロジックによる残像の徹底排除。
- **プロ仕様のWebダッシュボード**: Web Serial APIを活用し、ブラウザだけで測定・可視化・信号処理を完結。

## 🛠 ソフトウェア解説 (Firmware v0.18)

マイコン側のC言語プログラムは、リアルタイム性と測定精度を両立させるために以下の高度な制御を行っています。

### 1. 超高速PWM (187.5kHz)
PWM周波数を可聴域遥か上の187.5kHzに設定。これにより、安価なRCフィルタでもDC電圧の「波打ち（リップル）」を極限まで抑え込み、純粋なアナログ電圧を生成します。

### 2. ADC残像排除ロジック
チャンネル切り替え時の浮遊容量による影響を消すため、**「ダミーリード（空読み）」**を実装。
1. チャンネル切り替え
2. 1回目の変換（値を捨てる）
3. 2回目の変換（本番採用）
この手順により、VceとIcが互いに干渉することなく正確に測定されます。

### 3. 可変平均化フィルタ
`NUM_SAMPLES` 定数により、サンプリング回数を自由に変更可能（デフォルト16回）。ハードウェアで取り切れない微細なノイズをマイコン内部で処理します。

## 💻 UIダッシュボード解説 (Web Pro Edition)

HTML5 + Chart.jsで構築された専用の測定ソフトウェアです。

### 1. 1画面完結型 2カラム・レイアウト
- **左サイド**: デバイス制御、スムージング設定、メモ欄、シリアルモニターを配置。
- **右メイン**: 広大なグラフエリアを確保し、測定中に画面をスクロールする必要がありません。

### 2. インタラクティブ・スムージング
PC側のCPUパワーを活用した**「移動平均フィルタ」**を搭載。
スライダーを動かすだけで、生データ（Raw）からノイズを均した滑らかな曲線（Smooth）までリアルタイムに可視化モードを切り替えられます。

### 3. 統合シリアルモニター & メモ機能
マイコンからの生データをリアルタイムに監視できるシリアルモニターと、測定対象（例：2SC1815 Yランク）の情報を記録できるメモ欄を統合。測定データのスクリーンショット保存に最適です。

## 📐 回路コンセプト (Less is More)

- **Vce出力**: マイコンPWM端子 -> 1kΩ/0.1μF RCフィルタ -> トランジスタ

- **Ib制御**: マイコンPWM端子 -> RCフィルタ -> トランジスタ基板

オペアンプの位相補償や0V付近のオフセット問題を、ハードウェアを「引き算」し、ソフトウェアを「足し算」することで解決しました。
